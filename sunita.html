Sunita's Sewing and Alterations Visual Identity

I had the wonderful chance to help rebrand a local Ottawa business. Sunita's Sewing and Alterations is a home based studio for custom clothing and alterations. They recently created a new studio space for their work and wanted a new identity to go along with it. They were looking to add some class and youthfulness to their visual identity and tasked me with coming up with a new logo that was both current and timeless. I had a lot of fun incorporating the clien't personality into this brand. The end result was a handmade script for the main name and Futura, a timeless sans serif, for the tagline.

Both the client and myself were very happy with the end result. This was the first time i've made a handmade script for a client and it proved to very useful. I've also been tasked to create a new website for the them so that should be coming shortly!
